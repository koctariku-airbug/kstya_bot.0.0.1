# 🤖 bot_demo — Telegram Bot Project

Модульный Telegram-бот с разделением логики на handlers, services и utils. Проект сделан как основа для расширения и добавления новых функций без хаоса в коде.

⚡ Возможности
📩 Обработка команд через handlers
🧠 Разделение логики (services / utils)
⌨️ Поддержка клавиатур (keyboards)
📊 Логирование событий в logs/
📁 Готовая структура под масштабирование
🔧 Простое добавление новых функций
🧱 Архитектура проекта
bot_demo/
│
├── bot/
│   ├── data/            # хранение данных (файлы, состояния)
│   ├── handlers/        # обработчики команд и сообщений
│   ├── keyboards/       # inline / reply клавиатуры
│   ├── logs/            # лог-файлы бота
│   ├── services/        # бизнес-логика (основные функции)
│   ├── utils/           # вспомогательные функции
│   └── __pycache__/
🚀 Запуск проекта
cd bot_demo
python bot/bot.py

⚠️ Убедись, что установлен Python 3.10+

📦 Установка зависимостей

Если есть requirements.txt:

pip install -r requirements.txt
🧠 Как устроена логика
🔹 Handlers

Отвечают за входящие команды пользователя.

🔹 Services

Здесь находится вся “умная” логика (расчёты, обработка данных, API и т.д.)

🔹 Keyboards

Создание кнопок и интерфейса Telegram.

🔹 Utils

Вспомогательные функции (логирование, форматирование и т.п.)

🔹 Logs

Все события бота записываются сюда для отладки.

📊 Логирование

Бот сохраняет действия в папку:

bot/logs/

Там можно увидеть:

запуск бота
команды пользователей
ошибки
действия сервисов
